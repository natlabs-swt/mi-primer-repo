# Mi primer repositorio

Mi primera página web publicada con GitHub. Una tarjeta de presentación sencilla
con los colores de ShowerThinking.

## Qué hay aquí

- `index.html` — la página. Ábrela con doble clic para verla en tu navegador.
- `README.md` — este archivo. Es lo primero que se ve al entrar al repo.

## Cómo verla en vivo

Con GitHub Pages, esta página puede tener su propia URL pública. Pasos:

1. En el repo, ve a **Settings** → **Pages**.
2. En **Source**, elige la rama `main` y la carpeta `/ (root)`.
3. Pulsa **Save** y espera un minuto.
4. La URL aparecerá arriba: `https://tu-usuario.github.io/mi-primer-repo/`

## Cómo cambiar el texto

Edita `index.html` desde GitHub (icono del lápiz), modifica lo que quieras
entre las etiquetas, y haz **Commit changes**.
